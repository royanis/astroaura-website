# AstroAura - World's First Multilingual Astrology Super App

## About AstroAura
AstroAura is a production-ready, multilingual astrology and wellness mobile app available on iOS and Android. The app provides AI-powered cosmic insights, interactive birth charts, tarot readings, and personalized astrological guidance in 11 languages.

## Website Overview
This is the official marketing website for AstroAura, showcasing the app's features and providing download links to the mobile application.

## Key Features
- **Multilingual Support**: Available in 11 languages (English, German, French, Hindi, Chinese, Bengali, Tamil, Telugu, Kannada, Spanish, Portuguese)
- **AI-Powered Cosmic Chat**: Instant astrological guidance and personalized insights
- **Interactive Birth Charts**: Comprehensive natal chart analysis with cosmic journey experience
- **Day Planner**: Real-time planetary transit tracking with cosmic weather dashboard
- **Tarot Guidance**: Complete 78-card tarot deck with AI-powered interpretations
- **Astro Match**: Cosmic compatibility analysis for relationships
- **Cosmic Journal**: Personal wellness tracking with mood analytics
- **Cosmic Archetypes**: 11 unique celestial personality types
